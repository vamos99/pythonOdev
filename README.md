# Python Odevleri

Bu repo, Python temellerini pekistirmek icin hazirlanan iki konsol uygulamasini
icerir.

## Icerik

| Klasor | Kapsam |
| --- | --- |
| `Basic Bank/` | Hesap acma, para yatirma/cekme ve bakiye sorgulama |
| `To-Do List/` | Gorev ekleme, tamamlama, silme ve listeleme |

## Calistirma

```bash
python "Basic Bank/basic_bank.py"
python "To-Do List/todo_list.py"
```

## Notlar

- Uygulamalar egitim amaclidir ve kalici veritabani kullanmaz.
- Kodlar temel sinif yapisi, liste yonetimi ve kullanici girdisi pratiklerine
  odaklanir.
