# To-Do List

Basit bir komut satiri yapılacaklar listesi uygulamasidir. Kullanici gorev
ekleyebilir, gorevleri listeleyebilir, tamamlanmis olarak isaretleyebilir ve
silebilir.

## Calistirma

```bash
python todo_list.py
```

## Yapi

- `Task`: gorev adini ve tamamlanma durumunu tutar.
- `TaskManager`: gorev ekleme, silme, tamamlama ve listeleme islemlerini
  yonetir.

## Not

Bu calisma temel OOP, liste yonetimi ve kullanici girdisi pratikleri icin
hazirlanmistir. Kalici dosya/veritabani kullanmaz.
